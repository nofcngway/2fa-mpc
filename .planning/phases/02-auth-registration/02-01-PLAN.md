Define error variables (per D-10):
```go
var (
    ErrPasswordTooShort   = errors.New("password must be at least 12 characters")
    ErrMissingUppercase   = errors.New("password must contain at least one uppercase letter")
    ErrMissingLowercase   = errors.New("password must contain at least one lowercase letter")
    ErrMissingDigit       = errors.New("password must contain at least one digit")
    ErrMissingSpecialChar = errors.New("password must contain at least one special character")
    ErrSequentialChars    = errors.New("password must not contain 4 or more sequential characters")
    ErrRepeatedChars      = errors.New("password must not contain 4 or more identical consecutive characters")
)
```

Define `PasswordValidationError` struct (per D-03):
```go
type PasswordValidationError struct {
    Violations []error
}

func (e *PasswordValidationError) Error() string {
    msgs := make([]string, len(e.Violations))
    for i, v := range e.Violations {
        msgs[i] = v.Error()
    }
    return strings.Join(msgs, "; ")
}
```

Define `ValidatePassword(password string) error`:
1. Check `len(password) < 12` -> append ErrPasswordTooShort
2. Check no uppercase `[A-Z]` -> append ErrMissingUppercase
3. Check no lowercase `[a-z]` -> append ErrMissingLowercase
4. Check no digit `[0-9]` -> append ErrMissingDigit
5. Check no special char (anything not letter/digit) -> append ErrMissingSpecialChar
6. Call `containsSequential(password, 4)` -> if true, append ErrSequentialChars
7. Call `containsRepeated(password, 4)` -> if true, append ErrRepeatedChars
8. If `len(violations) > 0`, return `&PasswordValidationError{Violations: violations}`
9. Return nil

Define helper `containsSequential(password string, minLen int) bool`:
- Lowercase the password (per D-05)
- Define sequence strings (per D-01):
  ```go
  var sequences = []string{
      "abcdefghijklmnopqrstuvwxyz",
      "0123456789",
      "qwertyuiop",
      "asdfghjkl",
      "zxcvbnm",
      "7894561230",
  }
  ```
- For each sequence (and its reverse), use a sliding window of size `minLen` on the password. For each window of `minLen` consecutive password characters, check if that substring appears in the sequence or its reverse.
- Return true if any match found.

Define helper `containsRepeated(password string, minLen int) bool` (per D-02):
- Lowercase the password
- Slide a window of size `minLen`. If all characters in the window are identical, return true.

Define helper `reverse(s string) string`:
- Return the string reversed.
