        "fmt"

    **Sentinel error:**
    ```go
    var ErrSecretGeneration = errors.New("totp: failed to generate random secret")
    ```

    **hotp function (unexported, per D-04):**
    - Accept `secret []byte, counter uint64`, return `string`
    - Encode counter as 8-byte big-endian via `binary.BigEndian.PutUint64`
    - Compute HMAC-SHA1: `mac := hmac.New(sha1.New, secret)`, write counter bytes, `h := mac.Sum(nil)`
    - Dynamic truncation per RFC 4226 Section 5.4:
      - `offset := h[len(h)-1] & 0x0F`
      - Extract 4 bytes at offset: `uint32(h[offset])<<24 | uint32(h[offset+1])<<16 | uint32(h[offset+2])<<8 | uint32(h[offset+3])`
      - Mask: `truncated &= 0x7FFFFFFF`
      - `code := truncated % 1_000_000`
    - Return `fmt.Sprintf("%06d", code)` (zero-padded to 6 digits)

    **GenerateSecret:**
    - Allocate `make([]byte, 20)` for 20-byte secret
    - Fill with `io.ReadFull(rand.Reader, secret)` — return `nil, "", fmt.Errorf("%w: %v", ErrSecretGeneration, err)` on error
    - Encode: `base32.StdEncoding.WithPadding(base32.NoPadding).EncodeToString(secret)` — NO padding chars
    - Return `secret, encoded, nil`

    **GenerateOTP:**
    - `counter := uint64(unixTime) / 30`
    - Return `hotp(secret, counter)`

    **validateOTPAt (unexported):**
    - If `len(code) != 6`, return false immediately
    - `counter := uint64(unixTime) / 30`
    - Loop over `[]uint64{counter - 1, counter, counter + 1}`
    - If `hotp(secret, c) == code` for any c, return true
    - Return false

    **ValidateOTP (exported, per D-03):**
    - Return `validateOTPAt(secret, code, time.Now().Unix())`
