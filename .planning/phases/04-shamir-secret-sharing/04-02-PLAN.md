       )

    4. Implement `evalPolynomial(coeffs []byte, x byte) byte` (per D-10) — Horner's method in GF(256):
       - Start result = byte(0)
       - Iterate from highest degree down: result = gfAdd(gfMul(result, x), coeffs[i])
       - Return result

    5. Implement `lagrangeInterpolateAtZero(xs, ys []byte) byte` (per D-10):
       - For each point i, compute Lagrange basis L_i(0):
         - basis = byte(1)
         - For each j != i: basis = gfMul(basis, gfDiv(xs[j], gfAdd(xs[i], xs[j])))
         - Note: in GF(2^n), 0 - x_j = x_j (additive inverse is self, since XOR)
       - Accumulate: secret = gfAdd(secret, gfMul(ys[i], basis))

    6. Implement `Split(secret []byte, n, threshold int) ([]Share, error)` (per D-03):
       - Validate inputs per D-06: empty secret -> ErrEmptySecret, threshold < 2 -> ErrThresholdTooLow, n < threshold -> ErrNotEnoughShares, n > 255 -> ErrTooManyShares
       - Initialize n Share structs with Index = byte(i+1) (1-based per D-05), Data = make([]byte, len(secret))
       - For each byte of secret: build polynomial coeffs[0] = secretByte, coeffs[1..threshold-1] = random bytes from `crypto/rand`
       - Evaluate polynomial at each share index, store in share.Data[byteIdx]
       - Return shares

       - Check all share.Data same length -> ErrShareDataMismatch
       - For each byte index: extract xs (share indices) and ys (share data at byte index)
       - Call lagrangeInterpolateAtZero(xs, ys) for each byte
       - Return reconstructed secret
